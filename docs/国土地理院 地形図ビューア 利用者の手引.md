## 国土地理院 地形図ビューア 利用者の手引

### 1. 概要

本アプリケーションは、国土地理院の地図を背景に、お持ちのPNG画像を重ねて表示し、サイズや透明度を調整できるWebアプリケーションです。また、ExcelファイルからGPSデータを読み込み、地図上にマーカーとして表示することも可能です。

### 2. 機能説明

#### 2.1. 地図表示機能

アプリケーションを起動すると、初期地点として「箕面大滝」（緯度: 34.853667, 経度: 135.472041）を中心とした国土地理院の地図が、ズームレベル15で表示されます。地図の右下にはスケールバーが表示され、地図の縮尺を確認できます。

#### 2.2. 画像オーバーレイ機能

お手持ちのPNG画像を地図上に重ねて表示する機能です。

* **画像の読み込み**:
    1.  画面右上の `[画像読込]` ボタンをクリックします。
    2.  ファイル選択ダイアログが表示されるので、表示したいPNG画像ファイル（`image/png`）を選択します。
    3.  選択した画像が地図の中心に表示されます。

* **画像の調整**:
    * **表示倍率**: `表示倍率` の数値入力欄で、画像の表示サイズを調整できます。値を大きくすると画像が拡大され、小さくすると縮小されます。初期値は `0.3` で、入力範囲は0.1以上です。
    * **透過度(%)**: `透過度(%)` の数値入力欄で、画像の透明度を調整できます。`0` にすると完全な透明になり、`100` にすると完全に不透明になります。初期値は `50` で、入力範囲は0から100です。

* **画像のリセット**:
    * `[中心座標]` ボタンをクリックして中心座標設定モードに入ると、現在表示されている画像オーバーレイは地図から削除されます。

#### 2.3. GPSデータ読み込み機能

ExcelファイルからGPSデータを読み込み、地図上にマーカーとして表示する機能です。

* **GPSデータの読み込み**:
    1.  `[GPS値読込]` ボタンをクリックします。
    2.  ファイル選択ダイアログが表示されるので、GPSデータが含まれるExcelファイル（`.xlsx`形式）を選択します。
    3.  Excelファイルから読み込まれた地点情報が、地図上にマーカーとして表示されます。

* **マーカー情報の確認**:
    * 地図上のマーカーをクリックすると、その地点の名称がポップアップで表示されます。

* **Excelファイルの形式**:
    GPSデータを含むExcelファイルは、以下の形式である必要があります。
    * 最初のシートを使用します。
    * 1行目はヘッダーとして扱われ、2行目以降がデータとして処理されます。
    * 各列のデータは以下の通りです。
        * C列: 地点名の前半部分
        * D列: 緯度（度分秒形式の文字列） 例: "34502066" (34度50分20.66秒)
        * E列: 経度（度分秒形式の文字列） 例: "135274106" (135度27分41.06秒)
        * G列: 地点名の後半部分

#### 2.4. 中心座標の表示と設定機能

* **中心座標の表示**:
    * 画面右上の `北緯` と `東経` のテキストボックスに、現在地図上に表示されているマーカーの位置座標（緯度・経度）が小数点以下6桁で表示されます。

* **中心座標の設定**:
    1.  `[中心座標]` ボタンをクリックします。ボタンがアクティブ状態（押し込み状態）になります。
    2.  地図上の任意の位置をクリックします。
    3.  クリックした位置にマーカーが移動し、地図の中心もその位置に移動します。
    4.  `北緯` と `東経` の表示も更新されます。
    5.  中心座標設定モードは自動的に解除されます。
